RDA5820 超短波电台 (CLG 超短波电台)
基于 RDA5820 芯片的 Arduino 开源 FM 收发电台，支持半双工对讲、OLED 状态显示与按键调频，适合无线电爱好者学习与制作。
项目简介
本项目是一款低成本、开源的超短波（FM）对讲机，核心采用 RDA5820 FM 收发芯片，搭配 Arduino 主控，实现了完整的语音收发、频率调节、信号强度显示等功能。
项目作者：CLG 队伍
硬件平台：Arduino 系列单片机（兼容 STM32 等其他 MCU）
射频芯片：RDA5820（76–108 MHz FM 收发）
显示界面：0.96 寸 SSD1306 OLED 屏

去年刚接触编程时，我们因资金有限，买不起成品电台与闭源模块，却不愿让热爱止步。于是我从淘宝购入廉价的 RDA5820 模块，靠着自学啃透简陋的 IIC 驱动，逐行调试、反复校验，在无数个夜晚的摸索中，终于做出了属于自己的第一台电台。那种从无到有的成就感，让我更加坚定了深耕无线电 DIY 的决心。
可当我进阶挑战 BK4802 芯片时，却遭遇了最令人不齿的 “卡脖子”：开源作者突然删库跑路，关键资料一夜蒸发；模块商家紧握寄存器手册，只卖硬件拒绝提供编程接口；闲鱼卖家谈妥价格后，听闻我要源码便直接拉黑。被逼至绝境的我，只能抱着英文数据手册，一个寄存器、一个比特位地硬啃，硬生生靠自己的坚持完成了 BK4802 驱动的自主开发。我深恶痛绝这种删库跑路、收费锁码、技术垄断、割韭菜的行为 —— 它们违背了无线电 “开放互助” 的精神，更玷污了开源的初心。
